# Source Server Manager

Manage Source Engine dedicated servers on Windows through a simple interactive menu.
Install, configure, start, update and monitor your game servers — no advanced command-line knowledge required.

## Features

- **Background installation** — SteamCMD runs in a separate window while the main program stays fully responsive
- **Parallel installs** — download and install multiple servers simultaneously
- **Live status monitoring** — server state updates automatically on every menu action
- Automated install via SteamCMD with automatic download
- Start / stop / auto-restart with crash detection
- MetaMod:Source and SourceMod install, verification and removal
- SourceMod admin management from logs or connected players
- Windows Firewall rule management per server
- Context-aware menu — unavailable options are greyed out based on server state
- Multilingual interface (EN / IT)

## Requirements

- Windows 10 or 11
- PowerShell 5.1 (pre-installed on all modern Windows systems)
- Internet connection (for SteamCMD and server download)
- ~20 GB of free disk space per server

## Quick start

Double-click `start.bat`, or run from PowerShell:

```
.\start.bat
```

No additional software needed. SteamCMD is downloaded automatically on first use.
